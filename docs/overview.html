О проекте

Это кросс-платформенная haxe библиотека общего назначения.

Содержит невероятно удобные в использовании «кирпичи», которые работают одинаково на всех доступных хэксу платформах.

Включает в себя поддержку мощьнейшей событийной модели, которая включает все лучшие моменты различных платформ. Например обработчик события не обязан получать объект типа Event, как это навязывается в as3, он может просто принимать раздичные аргументы, но возможность принять событие Event для более гибкого контроля не куда не делась, при чём происходит автоматическое распознавание функции обработчика и вам не понадобится для этого лишний код. С вызовом события анологичная ситуация. По мимо этого имеется ещё куча вкусностей.
